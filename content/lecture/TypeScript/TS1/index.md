## 📑 타입스크립트?

자바스크립트는 현재 세가지 종류가 있어.
</br></br>

1. 웹브라우저에서 동작하는 표준 자바스크립트 ES5와
2. ESNext(ES6),
3. 그리고 타입을 추가한 타입스크립트임
   타입스크립트가 다 포함함.

```
function makePerson(name: String, age: number){}
```

이런게 타입스크립트</br>
자바스크립트에 강력한 Type기능을 추가해서 오류를 줄임.</br>
타입스크립트 소스코드는 TSC(타입스크립트 Compiler)를 통해 ES5자바스크립트로 변환됨.
